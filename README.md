# Pdf_reader_meaning_flutter
Smart Document Reader is a Python and Flutter based application that helps
users read PDF and Word files efficiently. It detects difficult words,
highlights them, and shows meanings with examples on hover or tap,
saving time and improving focus without switching to Google
